# Machine Learning Internship

This repository contains projects completed during my Machine Learning Internship. The tasks focus on applying machine learning and NLP techniques to solve real-world classification problems.



## Projects

### Credit Card Fraud Detection
- Detects fraudulent vs legitimate transactions  
- Uses transaction-based dataset  
- Models: Logistic Regression, Decision Tree, Random Forest  
- Evaluated using accuracy and confusion matrix  
- Handles imbalanced dataset  



### Customer Churn Prediction
- Predicts customer churn (leave/stay)  
- Uses behavioral & demographic data  
- Models: Logistic Regression, Random Forest, Gradient Boosting  
- Helps identify at-risk customers  



### SMS Spam Classification
- Classifies SMS as spam or legitimate  
- Uses NLP + TF-IDF/embeddings  
- Models: Naive Bayes, Logistic Regression, SVM  
- Automates spam detection
